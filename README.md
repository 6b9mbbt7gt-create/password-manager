## 📘 このアプリについて

このパスワード管理ツールは、**1Password** や **ID Manager** といった既存のパスワード管理ソフトを参考にしながら、  
自分自身の学習の一環として開発したものです。

「フォルダ階層でパスワードを整理できる仕組み」や  
「シンプルで扱いやすい UI」など、  
日常的に使いやすいパスワード管理ツールを目指して設計しました。

また、アプリ開発の基礎から GUI、データベース、アーキテクチャ設計までを  
一通り学ぶことを目的としており、  
**PySide6 を用いた GUI 開発** や  
**SQLite を使ったローカルデータ管理** の理解を深めるための実践的なプロジェクトでもあります。

学習目的で作成したツールではありますが、  
今後も機能追加や UI 改善を行い、  
より実用的なアプリケーションへと育てていく予定です。
